jquery 问题

  1. Q:IE、Chrome input file 文件名丢失问题 D:input file标签时，使用$.clone方法进行复制时发现文件名丢失。 A:由于浏览器安全考虑无法完整克隆元素完整信息 ：stackoverflow

  2. Q:$.data()方法取boolean值 D:标签<a data-is="false"/>;使用$.data("is")=="fasle"判断错误。 A:typeof($.data("is"))为boolean,不是字符串，所以需写成if($.data("is"))。

ajaxfileupload问题

  1. Q:ajaxfileupload返回json带<pre>

    D:文件正常提交，后台接收正常，action中返回json正常，用火狐查看，响应的值也是正常的,但是，ajax调用js中，就是不走success的方法。经过调查，是因为返回的json中，带有<pre>标签。 A:ajaxfileupload.js文件，uploadHttpData中eval( “data = ” + data );修改为data = jQuery.parseJSON(jQuery(data).text());
