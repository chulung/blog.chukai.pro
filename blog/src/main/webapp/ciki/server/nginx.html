异常

  • shared zone "SSL" has no equal addresses: 033F0000 vs 03330000.

windows下配置https需移除 nginx.conf 中此参数

ssl_session_cache    shared:SSL:1m;

The cache and other modules which require shared memory support do not work on Windows Vista and later versions due to address space layout randomization being enabled in these Windows versions. ref:known_issues

  • nginx: [emerg] bind() to 0.0.0.0:443 failed (10013: An attempt was made to access a socket in a way forbidden by its access permissions).

这是由于其他进程占用了nginx 的端口。 运行 cmd, 输入netstat -aon|findstr "443" 找到 0.0.0.0:443，找到 PID，在任务管理器结束进程。 vmware-hostd.exe
